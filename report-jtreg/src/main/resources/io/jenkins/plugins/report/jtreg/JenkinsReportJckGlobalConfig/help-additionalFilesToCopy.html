Specify the additional files which should be copied. By default, only files generated by this plugin are exported.

This field accepts a comma-separated list of files. Note that build.xml may not yet exist.

Examples:

  • ../../config.xml,build.xml - some relative files from buildrtoot
  • archive/reports - copy some artifacts from buildroot
  • /backup,/archive - copy two absolute files (dirs are not supported)

Leave empty if no additional files need to be copied.
