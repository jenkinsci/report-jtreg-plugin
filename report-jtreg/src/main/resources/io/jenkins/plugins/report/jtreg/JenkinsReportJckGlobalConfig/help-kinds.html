Specify the output formats for exported report files.

This field accepts a comma-separated list of writer kinds that determine the format of exported report files. Valid values are: PLAIN, JSON, and PROPERTIES.

Valid values:

  • PLAIN - Plain text format
  • JSON - JSON format
  • PROPERTIES - Java properties format

Examples:

  • PLAIN - Export only in plain text format
  • JSON, PROPERTIES - Export in both JSON and properties formats
  • PLAIN, JSON, PROPERTIES - Export in all three formats

Values are case-insensitive and whitespace around commas is ignored. Duplicate values will trigger a warning but are allowed.

Leave empty if no specific format is required (defaults will be used).
